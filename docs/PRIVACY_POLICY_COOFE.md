# Politica de Privacidade - Coofe

Data de vigencia: 2026-04-14

Esta Politica de Privacidade explica como o Coofe coleta, usa e protege informacoes quando voce usa o aplicativo.

## Quem Somos

Coofe e um aplicativo gratuito de produtividade gamificada para organizar projetos, rotinas, tarefas, objetivos, historico, relatorios e perfil.

## Informacoes Coletadas

O Coofe pode armazenar as seguintes informacoes fornecidas ou geradas pelo usuario:

- dados de conta, como e-mail usado para login;
- projetos, tarefas, rotinas, objetivos, notas e historico;
- configuracoes de perfil, progresso, moedas, energia, conquistas e preferencias;
- nome de usuario publico, quando configurado;
- dados sociais dentro do app, como amizades ou convites, quando usados;
- chave de API Groq, somente se o usuario informar voluntariamente para recursos de IA;
- configuracoes de notificacao, como lembretes e horarios escolhidos.

## Como Usamos as Informacoes

As informacoes sao usadas para:

- autenticar o usuario;
- salvar e sincronizar os dados do app;
- exibir progresso, historico, recompensas e relatorios;
- enviar notificacoes locais configuradas pelo usuario;
- gerar sugestoes e missoes com IA quando o usuario configura uma chave de API;
- manter recursos sociais opcionais do app.

## Notificacoes

O Coofe usa notificacoes locais para lembrar tarefas, rotinas e fechamento do dia quando o usuario ativa esse recurso. O usuario pode desativar notificacoes nas configuracoes do app ou nas configuracoes do sistema operacional.

## Compartilhamento de Dados

O Coofe nao vende dados pessoais.

Dados podem ser processados por servicos usados pelo app:

- Supabase, para autenticacao, banco de dados e sincronizacao;
- Groq, somente quando o usuario informa uma chave de API e usa recursos de IA.

## Armazenamento e Seguranca

Os dados sao armazenados usando Supabase e mecanismos locais do aplicativo. O Coofe usa medidas razoaveis para proteger as informacoes, mas nenhum sistema e completamente imune a falhas.

## Retencao e Exclusao

O usuario pode apagar dados pelo proprio app quando os recursos estiverem disponiveis ou solicitar exclusao dos dados pelo canal de contato informado nesta politica.

## Criancas

O Coofe nao e direcionado a criancas menores de 13 anos.

## Alteracoes Nesta Politica

Esta politica pode ser atualizada. A data de vigencia sera alterada quando houver mudancas relevantes.
